You are building BioFusion AI — a complete full-stack web application 
for real-time and file-based multimodal biosignal analysis using 
ECG, EMG, and EEG signals with AI-powered diagnostics.

This is NOT a Streamlit app. Build a proper full-stack web application:
- Backend: Python FastAPI
- Frontend: React.js with TailwindCSS
- Communication: REST API + WebSocket for real-time streaming
- Charts: Recharts + Plotly.js
- Theme: Dark medical/clinical aesthetic

── main.py ──
FastAPI app with:
- CORS middleware (allow frontend localhost:5173)
- WebSocket endpoint for real-time streaming
- Static file serving
- Lifespan event: load all ML models on startup

Router mounts:
app.include_router(upload.router, prefix="/api/upload")
app.include_router(analysis.router, prefix="/api/analysis")
app.include_router(hardware.router, prefix="/api/hardware")
app.include_router(ai_insights.router, prefix="/api/ai")

WebSocket endpoint:
@app.websocket("/ws/stream")
async def stream_websocket(websocket: WebSocket):
  Handles real-time ESP32 data streaming
  Sends JSON every 100ms:
  {
    "ecg": [...500 samples...],
    "emg": [...500 samples...],
    "eeg": [...250 samples...],
    "features": {...},
    "predictions": {...},
    "fusion": {...}
  }

── routers/upload.py ──

POST /api/upload/ecg
  Accept: .dat + .hea file pair OR .csv
  Parser:
    If .dat/.hea → use wfdb.rdrecord() to read
    If .csv → pandas read_csv, detect signal column
  Returns: 
  {
    "signal": [...normalized samples...],
    "sample_rate": 360,
    "duration_seconds": 30,
    "num_samples": 10800,
    "signal_type": "ECG"
  }

POST /api/upload/emg
  Accept: .csv (emg_healthy, emg_myopathy, emg_neuropathy)
  OR .dat/.hea pair
  Parse with pandas or wfdb
  Returns signal array + metadata
  
  IMPORTANT: The EMG CSV files have specific formats:
  emg_healthy.csv: tab-separated, multiple columns
  emg_myopathy.csv: same format
  emg_neuropathy.csv: same format
  Auto-detect separator, use first signal column
  Label: extract condition from filename 
  (healthy/myopathy/neuropathy)

POST /api/upload/eeg
  Accept: .edf files (chb01_01.edf etc.)
  Parse with: pyedflib or mne
  import mne; raw = mne.io.read_raw_edf(file_path)
  Extract first channel as primary EEG signal
  Returns signal array + sample rate

POST /api/upload/detect
  Auto-detect file type from extension + content
  Route to correct parser above
  Useful for drag-drop upload of any file

── core/ecg_model.py ──

class ECGModel:
  
  Training data: MIT-BIH Arrhythmia Database
  Use wfdb to read all 48 records
  
  Feature extraction per beat:
  [hr_bpm, mean_rr, std_rr, rmssd, pnn50, qrs_width,
   lf_power, hf_power, lf_hf_ratio]
  
  Class labels:
  N → Normal (class 0)
  V → PVC (class 1)
  A → Atrial ectopic (class 2)
  L, R, B → Bundle branch (class 3)
  
  Model: RandomForestClassifier(
    n_estimators=300,
    max_depth=20,
    class_weight='balanced',
    n_jobs=-1
  )
  
  Apply SMOTE for class balancing
  
  Target: >88% weighted F1 score
  
  Save to: models_saved/ecg_model.pkl
  Save scaler: models_saved/ecg_scaler.pkl

── core/emg_model.py ──

class EMGModel:
  
  TWO separate classifiers:
  
  1. Gesture Classifier (SVM):
     Features: [rms, mav, zcr, wl, ssc]
     Classes: Rest, Fist, Open, Point
     Trained on: synthetic + self-recorded data
  
  2. Condition Classifier (Random Forest):
     Features: [rms, mav, zcr, wl, mnf, mdf, var]
     Classes: Healthy, Myopathy, Neuropathy
     Trained on: emg_healthy.csv, emg_myopathy.csv,
                 emg_neuropathy.csv
     
     IMPORTANT: Load all three CSV files from data/emg/
     emg_healthy.csv → label 0 (Healthy)
     emg_myopathy.csv → label 1 (Myopathy)
     emg_neuropathy.csv → label 2 (Neuropathy)
     
     Extract features from each 500ms window
     Train RandomForest + SVM, use best
     Target: >90% accuracy (these are clean datasets)
  
  Save both models to models_saved/

── core/eeg_model.py ──

class EEGModel:
  
  Training data: CHB-MIT .edf files
  Use mne to read: mne.io.read_raw_edf()
  
  Seizure detection (binary):
  Normal vs Seizure segments
  
  Features per 1-second window:
  [delta_rel, theta_rel, alpha_rel, beta_rel,
   alpha_beta_ratio, spectral_entropy, engagement_idx]
  
  Model: SVC(kernel='rbf', C=10, probability=True)
  
  Target: >87% accuracy
  
  Mental state (rule-based, no training needed):
  alpha_beta_ratio > 1.5 → Relaxed
  alpha_beta_ratio < 0.8 → Focused
  theta_rel > 0.25 → Drowsy
  else → Neutral

═══════════════════════════════════════════════════════════
SUPPORTED FILE FORMATS — EXACT PARSING REQUIREMENTS
═══════════════════════════════════════════════════════════

ECG files (MIT-BIH format):
  Files: 100.dat + 100.hea (must upload both together)
  Parser: wfdb.rdrecord('data/mitdb/100')
  Signal: record.p_signal[:,0] (first channel = MLII lead)
  Sample rate: 360 Hz
  
EEG files (EDF format):
  Files: chb01_01.edf, chb01_02.edf, chb01_03.edf, chb01_04.edf
  Parser: mne.io.read_raw_edf(filepath, preload=True)
  Signal: raw.get_data()[0] (first channel)
  Sample rate: raw.info['sfreq'] (usually 256 Hz)
  
EMG files (CSV format from Physionet):
  Files: emg_healthy.csv, emg_myopathy.csv, emg_neuropathy.csv
  Parser: pd.read_csv(filepath, sep='\t') or sep=','
          (auto-detect separator)
  Signal: df.iloc[:, 0].values (first column)
  Sample rate: 4000 Hz (Physionet EMG dataset standard)
  Downsample to 1000 Hz for processing:
  from scipy.signal import decimate
  signal = decimate(signal, q=4)

DEMO MODE signal generation:
  ECG demo: neurokit2.ecg_simulate(duration=30, 
             sampling_rate=500, heart_rate=72)
  EMG demo: Random bursts using numpy
  EEG demo: Sum of sinusoids at band frequencies:
    alpha = 0.5 * np.sin(2*pi*10*t)  (10 Hz)
    beta = 0.2 * np.sin(2*pi*20*t)   (20 Hz)
    noise = 0.1 * np.random.randn(len(t))
    eeg = alpha + beta + noise
