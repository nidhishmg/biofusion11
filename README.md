# BioFusion AI

## Important: Model Files and Datasets

- Large model files (such as `backend/models_saved/ecg_model.pkl`) and datasets are **not included in this repository** due to GitHub file size limits.
- These files are stored locally on the developer's laptop.
- If you need the models or datasets, please contact the maintainer or use the provided download script (if available).

## How to Run

1. Clone the repository.
2. Install dependencies for backend and frontend.
3. Place required model files in `backend/models_saved/` as described above.
4. Start backend and frontend servers as per instructions in the respective folders.
